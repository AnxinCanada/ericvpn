Not working / Not Connecting?

  1. Make sure no other VPN application is running (INCLUDING CHROME EXTENSIONS)
  2. If on a Chromebook, remove all other chrome extensions that are VPNs
  3. Try reconnecting
  4. Check if wifi without VPN being connected works
  5. If step 4 is true, then restart your device
